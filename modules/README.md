Cyberinfrastructure Modules
===========================

To facilitate students' development of cyberinfrastructure skills aligned with
the course goals of CHE 331, including

- Basic Python syntax and control structures
- Using and writing Python functions to automate tasks
- Using the SymPy library as a symbolic mathematics engine to solve biophysical
chemistry problems
- & more!
